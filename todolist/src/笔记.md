# 生命周期函数
`componentWillMount`在组件挂载到页面前, 自动执行
`componentDidMount`在组件挂载到页面后，自动执行

## 当组件实例被创建并插入 DOM 中时，其生命周期调用顺序如下：

1. constructor()
2. static getDerivedStateFromProps()
3. render()
4. componentDidMount()

## redux
1. createStore
2. store.dispatch
3. store.getState
4. store.subscribe
